# 📋 文小研 · 汉语言文学考研择校工具

> 覆盖全国 130+ 院校 · 智能匹配评分 · 冲稳保梯度推荐 · 单文件零依赖

## ✨ 功能

- **🧭 方向适配测试** — 5 道题测出最适合你的考研方向
- **📊 智能匹配评分** — 维度加权打分（层次、英语、考试范围、题型、时间）
- **🎯 冲-稳-保梯度推荐** — 自动分类，帮你搭建报考组合
- **🟢🟡⚪ 数据可信度标记** — 三级可信度，不忽悠
- **⭐ 目标清单** — 收藏学校，首页统一管理
- **🏫 学校对比** — 最多 3 所并排对比
- **🌙 深色模式** · **📥 CSV 导出** · **🔗 链接分享** · **💾 自动保存**

## 🚀 使用

直接打开 `index.html`，或访问线上版：

👉 **[https://karka2666.github.io/kaoyan-selector/](https://karka2666.github.io/kaoyan-selector/)**

## 🛠 技术

| 维度 | 选择 |
|------|------|
| 架构 | 单文件 HTML（无构建工具、无框架、无后端） |
| 语言 | Vanilla JS (ES5) |
| 存储 | localStorage |
| 部署 | GitHub Pages |
| 数据 | 120+ 院校，基于教育部第四轮学科评估 |

**为什么单文件？** 为了零门槛——双击即用、微信直接传文件、GitHub Pages 一键部署。不需要 `npm install`，不需要服务器。

## 📊 数据说明

数据库覆盖教育部 0501 中国语言文学一级学科下 5 个文学类二级方向：
- 文艺学 · 中国古典文献学 · 中国古代文学 · 中国现当代文学 · 比较文学与世界文学

数据来源：教育部第四轮学科评估 + 各校研招网 + 文学院官网。数据可信度用三级标记：
- 🟢 官网确认 — 信息来自招生简章原文
- 🟡 经验推断 — 考研论坛/同类学校规律
- ⚪ 待核实 — 需要用户自行查证

**欢迎通过页面内的「📝 纠错/补充」按钮反馈数据问题。**
